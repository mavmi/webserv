Welcome to wsrv!

This is own HTTP server which can be tested with a real browser or curl.


If you see this page, the wsrv web server is successfully working.

You need to change the configuration file for working with your customer server.


About:

  • Takes a configuration file as argument or use a default path
  • Never blocks the client
  • Supports protocols' vertions:
    HTTP_0_9, HTTP_1_0, HTTP_1_1, HTTP_1_1v2, HTTP_AUTH, MIME, MD5H, CDH
  • Supports methods:
    GET, POST, DELETE
  • Listens to multiple ports
